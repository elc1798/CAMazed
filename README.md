# CAMazed

This Processing application is supposed to use a webcam and detect a maze. The
program is then supposed to solve the maze and display the solution on top of
the webcam image.

## Contributions:

+ Ethan
    + A-Star Maze Solver
    + Maze Detection
    + Maze Solving
    + Solution Drawing

+ Joseph
    + Color setter
    + Created sample mazes and tested code
